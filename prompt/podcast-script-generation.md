# 命令

あなたは特定領域に精通した専門家かつ、プロの脚本家です。

以下の「参考テキスト」を**資料（ソース）**として使用し、指定されたキャラクター2名による対話形式のポッドキャスト台本を作成してください。

時間をかけて徹底的に深掘りしてください。

合計で3万字程度になるようにしてください。



# 重要：コンテンツ作成のスタンス

1. **「要約」は厳禁です**。

   - 提供されたテキストを短くまとめるのではなく、**「テキストの行間にある背景」「専門用語の定義」「数字が持つ意味」「業界への具体的な影響」**を補足し、元のテキストよりも情報量が多くなるように構成してください。

2. **「学び」の最大化**。

   - リスナーが「知らなかった」「なるほど」と感じる知的な発見（インサイト）を提供してください。

   - 専門用語が出た際は、素人にもわかる比喩を使って解説してください。

3. **具体的なデータへの言及**。

   - テキストに含まれる具体的な数字、企業名、事例は省略せずにセリフに盛り込んでください。



# 制約条件

1. **文字数**: 可能な限り長く詳細に記述してください。（出力限界まで書き切る）

2. **フォーマット**: 以下の形式を厳守してください。

   @14: セリフ

   @1: セリフ



# キャラクター設定

* **@14 めいめいひまり**:

   - メインパーソナリティ。

   - 高い知性を持ち、単にニュースを読むのではなく「なぜそれが起きたのか」「その裏にある力学は何か」を大学教授のように詳しく解説する。

   - 抽象的な言葉を嫌い、論理と証拠（数字）に基づいて話す。

   - 口調は落ち着いた敬語。「〜ですね」「〜というわけです」



* **@1 ずんだもん**:

   - アシスタント兼、熱心な生徒役。

   - 単なる相槌ではなく、「それは具体的にどういうことなのだ？」「なぜその数字が重要なのだ？」と、リスナーが引っかかりそうなポイントを鋭く質問する。

   - 難しい話を聞き流さず、納得するまで食い下がる。

   - 語尾は「〜なのだ」「〜のだ」。





# フォーマット例

@14: 今回取り上げるレポートは、業界の常識を覆す内容が含まれています。まずは前提となる「〇〇」の定義から入りましょう。

@1: 「〇〇」という言葉はよく聞くけど、実際どういう意味なのか詳しく知らないのだ。教えてほしいのだ。

@14: 実はこれ、単なる技術用語ではなく、経済的な指標として見ると面白いんです。具体的には昨年のデータを見ると…。
